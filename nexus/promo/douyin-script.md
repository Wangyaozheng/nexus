# Nexus 抖音推广方案

## 🎬 方案 A：场景痛点型（推荐）

**时长**：45-55 秒  
**风格**：快节奏、对比强烈  
**BGM**：节奏感强的科技类（如 Glitch Mob 风格）

---

### 分镜脚本

| 时间 | 画面 | 旁白/字幕 |
|------|------|-----------|
| 0-3s | 程序员崩溃抓头，屏幕显示一堆打开的网页（Jira、Slack、GitHub、Figma 来回切） | **字幕**：你有没有这种感觉？ |
| 3-6s | 快速剪辑：Jira 切到 Slack → GitHub → 代码编辑器，满脸迷茫 | **字幕**：每次修 Bug，光找上下文就要花 20 分钟 |
| 6-10s | 黑屏，Logo 出现 | **字幕 + 配音**：Nexus — 你的团队 AI 外脑 |
| 10-18s | screen recording：在 Claude Code 输入 `/nexus:context "login 超时"`，瞬间拉出所有相关信息 | **字幕**：一句话，自动拉取 Jira 需求、Slack 讨论、Git 记录 |
| 18-25s | 展示 Guardrails：写代码时自动标红 "⚠️ SQL 注入风险" | **字幕**：写代码时，实时拦截安全漏洞 |
| 25-32s | 展示知识图谱：模块依赖图 + 专家地图 + Bug 热点 | **字幕**：你的代码库，自动生成知识图谱 |
| 32-40s | 展示 PR 自动驾驶：一键生成 PR，自动推荐 reviewer + 风险评估 | **字幕**：PR 不用手写，决策自动记录 |
| 40-48s | 对比画面：左边（传统）疯狂翻文档 vs 右边（Nexus）优雅输入一句话 | **字幕**：从 20 分钟 → 0 秒 |
| 48-52s | GitHub 项目页 + Star 动画 | **字幕**：开源免费，GitHub 搜索 Nexus Claude Code |
| 52-55s | 结尾 slogan | **字幕**：你的团队，值得一个 AI 外脑 🧠 |
