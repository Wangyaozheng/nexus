# Changelog

## [1.0.0] — 2026-05-26

### Added
- Initial release of Nexus (formerly Team Mind)
- **Context Assembler**: Auto-pull context from Git, GitHub Issues, Jira, Slack
- **Guardrails Engine**: 7 built-in rules for security and code quality
- **Decision Logger**: ADR creation, listing, searching, and status management
- **Knowledge Graph**: Dependency analysis, expert map, hotspots, fragility tracking
- **PR Autopilot**: Smart PR generation with reviewer recommendation and risk analysis
- **Dashboard**: Team intelligence overview with metrics and trends
- Project initialization with auto-detection of tech stack
- Git post-commit hook for auto-updating knowledge graph
- Pure bash guardrails engine (no python dependency)
- Cross-platform temp file handling
- Environment-variable-driven credential management (no config file secrets)
